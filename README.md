# Parts Avatar Backend Technical Assessment

Welcome!

This assessment is designed to evaluate your proficiency in **Python**, **SQL**, and **Git**.

## Instructions
1. Create a virtual environment and install libraries from the requirements.txt. 
2. **Clone** this repository to your local machine, and create a new branch starting with your initials.
4. Run `assessment_db.py` to set up the db.
3. Complete the logic in the provided task files.
4. **Commit** your changes with a descriptive message per each task.
5. Once all tasks are done, pls update the requirements.txt synced with the virtual environment setup.
6. Notify the interviewer when you are done.

## Requirements
- Python 3.12+
- Basic knowledge of SQL syntax (SQLite dialect).
- Clean, readable code following PEP 8 standards.
